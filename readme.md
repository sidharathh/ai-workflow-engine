# 🚀 AI Workflow Engine

A full-stack AI-powered workflow automation platform that allows users to create and execute dynamic AI workflows like summarization and rewriting.

---

## 🔥 Features

- User authentication (JWT)
- Create dynamic workflows
- Execute AI workflows
- Gemini AI integration
- Execution history
- Secure APIs
- Responsive UI (React + Tailwind)

---

## 🧠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Atlas)
- JWT Authentication
